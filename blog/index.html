Cómo añadir nuevos artículos

  1. Crea el nuevo archivo dentro de /blog/.
  2. Añádelo en esta portada dentro del bloque “Últimos artículos”.
  3. Incluye un enlace hacia la home para que el blog empuje tráfico a la web principal.
  4. Actualiza el sitemap.xml para facilitar la indexación.

Siguiente paso recomendado

Publica estos tres artículos a la vez y enlázalos desde la portada principal o desde una sección visible del sitio. Eso ayuda a que Google descubra antes el blog y mejora la circulación interna del tráfico.
